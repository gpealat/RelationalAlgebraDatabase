# Relational Algebra Exercices
Small project to test using relational algebra to write SQL queries

## How to initialize the DB?
python initialize_db.py

## How to test a query?
run the command: python requests.python
Then you will be prompted to enter your SQL query (i.e. SELECT * FROM ACTORS)

## How to change the data?
The data are saved in./data folder. You can change and add your own data.
After, you need to reinitialize the database

## How to change the default setup
The SETUP.INI file holds all the parameters that can be changed
